Project 1:Inventory Management System

This is a specialized inventory management solution designed to streamline the lifecycle of footwear retail. From tracking the latest sneaker drops to managing daily sales, this system provides real-time visibility into stock levels, sales performance, and procurement needs.

Funcionalities:

  • Smart Stock Tracking: Organize inventory by Model, Brand, Size (US/UK/EU), and Colorway. Never lose track of a specific pair again.
  • Buy/Sell Transaction Engine: * Inward (Buy): Easily log new stock arrivals from wholesalers or individual sellers, including purchase price and date.
  • Real-time Analytics: A dynamic dashboard that calculates total profit margins, identifies "dead stock" (slow-moving items), and highlights your best-selling sizes and brands.

Technologies Used:

  • c#
  • mysql
  • html
  • css
  • javascript
